AppGenerator¶

The AppGenerator Module performs a complete deployment test on 200 applications over 40 devices.

Usage:

python3 AppGenerator.py
AppGenerator.main()¶

Main function to run the application generator.

This function parses command line arguments, loads configuration, generates application lists, and exports the applications to the specified output file.

AppGenerator.parse_args()¶

Parses command line arguments for the application generator.

Returns:

Parsed command line arguments.

Return type:

argparse.Namespace
